What it does

Tools for using AI with intention

Chrome Extension

Think before you ask

Intercepts prompts on ChatGPT and asks if this is something you should try yourself first. Classifies each prompt as a thinking task or a lookup task using the Anthropic API.

Desktop App

Time tracking for AI apps

A macOS menu bar app that tracks how long you spend in ChatGPT, Claude, Copilot, Notion, and Perplexity each day. Sends a notification when you hit your self-set limit.

AI Literacy Quiz

Measure what you actually know

A short quiz that tests whether you can answer your own questions independently. Gives you a score and tells you which areas you rely on AI most.

Usage Logger

Log AI interactions manually

A simple form to log each time you use AI, what you used it for, and whether you could have done it yourself. No accounts, no sync, stored in your browser.

Chrome Extension

Think before you send

When you type a prompt into ChatGPT and press send, the extension intercepts it, classifies it, and asks if it's worth trying yourself first. Lookup tasks go straight through. No interruption.

Download extension (.zip)
1. Download and unzip the file above.
2. Open Chrome and go to chrome://extensions
3. Enable Developer Mode (top right toggle).
4. Click Load Unpacked and select the unzipped folder.
5. Click the extension icon, then Settings, and paste your Anthropic API key.
AI Literacy Score

Do you know what you know?

The quiz asks you ten questions across categories like writing, coding, research, and decision-making. It scores how often you could answer them without AI help.
