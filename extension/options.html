Privacy & Data
  • Your API key is stored only in your browser via chrome.storage.local — never uploaded anywhere.
  • Only the text of your prompt is sent to the Anthropic API for classification. No site data, cookies, or identity.
  • The actual prompt content is never stored by this extension — only the result (THINKING or LOOKUP).
  • All your stats and history live entirely on your device.
